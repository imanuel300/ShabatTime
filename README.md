## שימוש

### בחירת עיר
יש שתי דרכים לבחור עיר:
1. בחירה מהרשימה הנפתחת באתר
2. שימוש בפרמטר URL:
   - הוספת `?city=CITY_NAME` לכתובת האתר
   - לדוגמה:
     - `index.html?city=Jerusalem` - לירושלים
     - `index.html?city=Tel%20Aviv` - לתל אביב
     - `index.html?city=Haifa` - לחיפה
